## IPFSDictChain

**IPFSDictChain: Python Dicts That Live on IPFS**

Tired of saving data that only exists on one machine? With IPFSDictChain, your Python dictionaries become portable, tamper-proof, and globally accessible—thanks to IPFS.

### It's just like using a regular dict, but smarter:
- Add `.save()` to push your data to IPFS instantly
- Use `.load(cid)` to bring it back—anywhere, anytime
- Every version is content-addressed, so nothing gets lost or overwritten

### Perfect for:
- Sharing config or state across decentralized apps
- Storing experiment results, backups, or metadata
- Building trustless audit trails or immutable logs

No need to reinvent the wheel. IPFSDictChain gives you the tools to build persistent, distributed data flows—right from your Python shell.

**Persistence, meet simplicity.**

**Links:**
- GitHub: https://github.com/ValyrianTech/ipfs_dict_chain

## Valyrian Spellbook

Valyrian Spellbook is an Open Source framework that combines the power of Artificial Intelligence and Blockchain technology, allowing you to create the back end of your applications with ease.

Whether you're an experienced developer or have no coding skills at all, Valyrian Spellbook lets you build your applications in a fraction of the time it would take to create them from scratch.

### Features:
- Configure a REST API server with triggers and actions to automate your transactions
- Set up triggers to execute custom python scripts when certain conditions are met
- Define actions to be taken automatically when a trigger is activated (e.g., sending a Bitcoin transaction)
- User-friendly interface to set parameters without coding
- Write custom python scripts for more complex applications

In summary, Valyrian Spellbook is a powerful tool that simplifies the process of building applications powered by AI and Blockchain technology, allowing you to create custom solutions in a matter of hours.
